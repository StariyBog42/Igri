Предварительные требования
Установленная Java 11 или выше

IntelliJ IDEA или другая Java IDE

Внешние библиотеки:

sqlite-jdbc-3.42.0.0.jar

jfreechart-1.5.4.jar

jcommon-1.0.23.jar

Шаги запуска
Клонируйте репозиторий или скопируйте файлы проекта

Откройте проект в IntelliJ IDEA

Добавьте JAR файлы в проект (File → Project Structure → Libraries)

Поместите CSV файл в папку resources/

Запустите класс Main.java
